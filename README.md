# descripttrashout
## 1. Экономическая целесообразность
Проект рентабелен благодаря низким операционным затратам на запуск (MVP на базе Telegram). Комиссия платформы в 20% покрывает расходы на поддержку, верификацию и развитие, обеспечивая прибыль даже при умеренном объеме заказов. Отсутствие необходимости в собственном автопарке и штатных грузчиках переводит основные издержки в переменные, что минимизирует финансовые риски.

## 2. Сезонность
Одним из лучших периодов для работы сервиса является лето. Это связано с тем, что летом у школьников-подростков гораздо больше свободного времени благодаря каникулам. Также лето - это сезон отпусков и благоприятные условия для проведения ремонта, что увеличивает количество заказов на вывоз строительного мусора.

## 3. Оплата труда
Оплата исполнителям (Оптимальная сумма которая будет найдена позже) установлена на уровне рыночных ставок курьерских служб, но выигрывает в эффективности за счет отсутствия логистического плеча. Подросток не тратит время и деньги на дорогу, работая в своем доме или районе, поэтому получает вознаграждение за 100% рабочего времени.

## 4. Ниша выноса строительного мусора
Вынос строительного мусора через бот целесообразнее, чем использование труда профессиональных строителей. Квалифицированные бригады теряют деньги на простых погрузочных работах, тогда как для подростков это доступный физический труд без необходимости спецподготовки. Это позволяет занимать свободную нишу «легкого аутсорса» в ремонте, где клиент платит только за время и усилия, а не за квалификацию мастера.
